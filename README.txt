PUBG UC Purchase Bot
===================

Bu loyiha Telegram orqali PUBG UC sotib olish va boshqarish uchun ishlab chiqilgan botni o‘z ichiga oladi. Bot polling usuli yordamida ishlaydi va foydalanuvchilarga UC xarid qilish, bonus olish va pulni chiqarish imkoniyatini beradi.

Xususiyatlar
------------
- Til tanlash (o‘zbekcha, ruscha, inglizcha)
- UC xarid qilish (turli miqdorlarda)
- Welcome bonus (300 UC)
- Referal tizimi
- UC o‘tkazish va pulni chiqarish funksiyalari

O'rnatish Ko'rsatmalari
-----------------------
1. Kerakli kutubxonlarni o'rnatish:
   - `pip install python-telegram-bot`

2. Kodni ishga tushirish:
   - `python bot.py` buyruqni ishga tushiring.

Fayllar
-------
- `bot.py`: Asosiy bot kodi.

Qo'llanma
---------
- Botni ishga tushirishdan oldin `BOT_TOKEN` qiymatini BotFather’dan olingan token bilan almashtiring.
- `ADMIN_CHAT_ID` qiymatini adminingizning Telegram ID’si bilan yangilang.
- Botni sinovdan o‘tkazish uchun Telegram’da `/start` komandasini yuboring.

Sinov
-----
- Botni ishga tushiring va tilni tanlang.
- Asosiy menyudan turli funksiyalarni sinab ko‘ring (UC olish, bonus, referal va boshqalar).

Muallif
-------
- Loyiha [sizning ismingizni qo‘shishingiz mumkin] tomonidan yaratilgan.
- Bugungi sana: 25.07.2025
